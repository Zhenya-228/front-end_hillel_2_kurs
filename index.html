Нотатки

  • git status - переглянути стан або зміни файлів
  • git --version - переглянути версію гіта
  • git log - переглянути перелік всіх комітів та користувачів які комітили
  • git add . - додати нові файли(зміни) для коміту
  • git commit -m "" - закомітити файли локально(вони є тільки на компі та і гіті на будуть показуватися)
  • git push origin (назва гілки або HEAD (якшо ту у якій ти знаходишся)) - закомітити файли до свого гіту
  • git push (куди відправляти) (назва гілки) - закомітити файли до гіту
  • git branch - переглянути у якій гілці ти знаходишся
  • git checkout -b (назва гілки) - створити або переключитися на іншу гілку
  • git pull origin (назва гілки) - отримати із свого сервера нові зміни
  • git pull (звідки брати) (назва гілки) - отримати із сервера нові зміни
  • git remote -v - переглянути список налаштованих віддалених сховищ
